### 📦 의존성 패키지 설치 및 실행

의존성 설치 후 core 프로젝트 실행합니다.
이후 각각의 프로젝트를 실행 하면 됩니다.

```bash
# 모든 프로젝트 의존성 설치
npm install:all

# core 폴더 프로젝트 실행
npm run start:core

# 각각의 프로젝트 실행
npm run start:hacker
npm run start:company
npm run start:findthegap
```
